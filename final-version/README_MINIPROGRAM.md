# 火电机组考核系统 - 微信小程序版本

## 📖 项目说明

这是火电机组考核系统的微信小程序版本，用于解决微信访问限制问题，提供更好的用户体验。

## 📁 项目结构

```
miniprogram/                    # 小程序根目录
├── pages/                      # 页面目录
│   ├── index/                  # 首页（角色选择）✅
│   ├── student/                # 学生端
│   │   ├── login/             # 登录页 ✅
│   │   ├── exam/              # 答题页 ⏳
│   │   └── result/            # 成绩页 ⏳
│   └── teacher/               # 教师端
│       ├── login/             # 登录页 ⏳
│       ├── stats/             # 统计页 ⏳
│       └── chapter/           # 章节出题页 ⏳
├── components/                 # 公共组件 ⏳
├── utils/                      # 工具类 ✅
│   ├── request.js             # API请求封装 ✅
│   └── util.js                # 工具函数 ✅
├── assets/                     # 资源文件 ⏳
│   └── icons/                 # 图标资源 ⏳
├── app.js                      # 小程序入口 ✅
├── app.json                    # 小程序配置 ✅
└── app.wxss                    # 全局样式 ✅

docs/                           # 文档目录
└── 微信小程序开发指南.md       # 详细部署文档 ✅
```

## 🚀 快速开始

### 1. 下载微信开发者工具
访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 导入项目
1. 打开微信开发者工具
2. 选择"导入项目"
3. 项目目录：选择 `miniprogram` 文件夹
4. AppID：填入你的小程序AppID（需要先注册）

### 3. 配置服务器域名
在小程序后台配置合法域名：
```
https://90c19216-7224-4e07-9c9b-2d1be18d1149.dev.coze.site
```

### 4. 开始开发
- 点击"编译"预览效果
- 点击"预览"生成二维码，真机测试
