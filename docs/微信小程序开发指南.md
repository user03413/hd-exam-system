function request(url, method = 'GET', data = {}) {
  return new Promise((resolve, reject) => {
    // 方式1: 使用Coze平台
    const fullUrl = app.globalData.baseUrl + url

  })

## ⚠️ 注意事项

### 1. 域名配置
- ✅ 必须在小程序后台配置合法域名
- ✅ 域名必须是HTTPS
- ✅ 域名必须已备案

### 2. 开发限制
- 小程序包大小限制：2MB
- 分包大小限制：每个分包2MB，总共20MB
- 网络请求超时：60秒

### 3. 审核注意事项
- 内容不能涉及政治敏感
- 不能有诱导分享
- 功能必须完整可用
- 必须提供测试账号
