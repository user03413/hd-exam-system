## 🚀 启动步骤

### 第一步：打开项目

1. 打开**微信开发者工具**
2. 点击「**导入项目**」或「**+**」
3. 选择 `miniprogram` 文件夹
4. AppID 选择「**测试号**」或填写 `touristappid`
5. 点击「**导入**」

### 第二步：关闭域名校验（必须！）

```
1. 点击右上角「详情」按钮
2. 选择「本地设置」标签
3. 勾选以下选项：
   ☑️ 不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书
```

### 第三步：编译运行

```
点击顶部「编译」按钮 或 按 Ctrl+B
```

### 第四步：测试登录

**教师端测试：**
```
1. 点击「教师管理」
2. 输入工号：654321
3. 点击「登录」
4. 预期：跳转到统计页面
```

**学生端测试：**
```
1. 点击「学生考试」
2. 输入学号：123456
```

## ⚠️ 常见问题解决

### 问题1：登录失败，提示 "Cannot read property 'globalData' of undefined"

**原因：** 缓存了旧代码

**解决：**
```
1. 点击菜单「工具」→「清除缓存」→「清除全部缓存」
2. 重新编译（Ctrl+B）
```

### 问题2：网络请求失败

**原因：** 域名校验未关闭

**解决：**
```
详情 → 本地设置 → 勾选「不校验合法域名...」
```

### 问题3：页面空白或样式错乱

**原因：** 文件缺失或路径错误

**解决：**
```
1. 检查 pages 目录下的文件是否完整
2. 检查 app.json 中的 pages 配置
```
