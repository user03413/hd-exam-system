## 🚀 如何使用小程序

### 第一步：注册小程序账号

**访问**：https://mp.weixin.qq.com/

1. 点击"立即注册" → 选择"小程序"
2. 填写邮箱、密码等信息
3. 邮箱激活
4. 信息登记（选择"个人"或"企业"）
5. 管理员扫码验证

**⏱️ 预计耗时**：10-15分钟

---

### 第二步：获取AppID

注册完成后：

1. 登录小程序后台
2. 开发 → 开发管理 → 开发设置
3. 复制"AppID"（如：wx1234567890abcdef）

---

### 第三步：配置服务器域名

**在小程序后台**：

1. 开发 → 开发管理 → 开发设置
2. 找到"服务器域名"
3. 点击"修改"
4. 添加"request合法域名"：

```
https://90c19216-7224-4e07-9c9b-2d1be18d1149.dev.coze.site
```

**⚠️ 重要提示**：
- 域名必须是HTTPS
- 域名必须已备案
- Coze开发域名可能无法通过审核，建议使用正式域名

---

### 第四步：下载开发者工具

**访问**：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

- 下载对应系统版本（Windows/Mac）
- 安装开发者工具

---

### 第五步：导入项目

1. 从GitHub下载项目代码
   ```bash
   git clone https://github.com/user03413/hd-exam-system.git
   ```

2. 打开微信开发者工具

3. 点击"+"号，选择"导入项目"

4. 填写信息：
   - **项目名称**：火电机组考核系统
   - **目录**：选择 `miniprogram` 文件夹
   - **AppID**：填入你的AppID
   - **开发模式**：小程序
   - **后端服务**：不使用云服务

5. 点击"导入"

---

### 第六步：预览测试

**在开发者工具中**：

1. 点击"编译"按钮
2. 选择不同设备型号预览
3. 测试所有功能

**真机测试**：

1. 点击"预览"按钮
2. 使用微信扫描二维码
3. 在真机上测试功能

---

### 第七步：上传审核

**在开发者工具中**：

1. 点击右上角"上传"按钮
2. 填写版本号：1.0.0
3. 填写项目备注：火电机组考核系统初始版本
4. 点击"上传"

**在小程序后台**：

1. 管理 → 版本管理
2. 开发版本 → 点击"提交审核"
3. 填写审核信息：
   - **功能页面**：选择首页
   - **测试账号**：
     - 学生账号：123456
     - 教师账号：654321
   - **功能介绍**：在线考试系统，支持学生答题、教师管理
4. 提交等待审核

**⏱️ 审核时间**：通常1-3个工作日

---

### 第八步：发布上线

审核通过后：

1. 版本管理 → 审核版本
2. 点击"发布"
3. 选择"全量发布"
4. 完成！
