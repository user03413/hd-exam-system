## 📊 数据来源

### 章节列表
- **来源**: `load_questions()` 函数读取的题库数据
- **统计方式**: 遍历所有题目，按 `chapter` 字段分组计数
- **排序**: 按章节名称排序

### 题库数据
- **位置**: `assets/questions.json` 或内存中的题库数据
- **字段**: 每道题包含 `chapter` 字段，标识所属章节
- **总数**: 244题，分布在17个章节
