## 使用流程

### 第一步：登录验证

1. 打开考试系统网址
2. 输入您的学号（如：220252216068）
3. 点击「验证并进入考试」按钮
4. 系统验证通过后显示您的姓名和专业信息

![登录页面示意]

### 第二步：开始答题

1. 确认个人信息无误
2. 阅读考试须知
3. 点击「开始答题」按钮
4. 系统随机抽取10道题目

![准备页面示意]

### 第三步：作答

#### 单选题
- 点击选项即可选择
- 只能选择一个答案
- 可随时修改选择

#### 多选题
- 点击选项进行多选
- 可选择多个答案
- 选中的选项会高亮显示

#### 简答题
- 在文本框中输入答案
- 支持多行输入
- 答案会自动保存

![答题页面示意]

### 第四步：提交答卷

1. 答完所有题目后点击「提交答卷」
2. 如有未作答题目会弹出提示
3. 确认后系统自动判分

### 第五步：查看结果

1. 查看总分和答题统计
2. 浏览每道题的详细解析
3. 点击「前沿拓展」查看相关技术资讯
4. 点击「导出学习报告」下载PDF报告

![结果页面示意]

## 考试规则

### 作答要求

1. **独立完成**：请在安静环境下独立完成考试
2. **认真审题**：仔细阅读题目和选项
3. **检查确认**：提交前请确认答案无误

### 评分标准

#### 单选题
- 完全正确：10分
- 错误：0分

#### 多选题
- 完全正确（所有选项都对）：10分
- 部分正确或错误：0分

#### 简答题
- 完全正确：10分
- 部分正确（关键词匹配≥60%）：10分
- 基本正确（关键词匹配≥30%）：5分
- 错误或未作答：0分

### 成绩等级

| 分数区间 | 等级 | 评语 |
|----------|------|------|
| 90-100分 | 优秀 | 优秀！继续保持！ |
| 80-89分 | 良好 | 良好，再接再厉！ |
| 60-79分 | 及格 | 及格，需要加强学习。 |
| 0-59分 | 不及格 | 不及格，请认真复习。 |

## 常见问题

### Q1：学号输入后提示"学号不存在"怎么办？

**A**：请检查以下几点：
1. 确认学号输入是否正确（12位数字）
2. 检查是否有多余空格
3. 确认您的学号是否在学生名单中
4. 如仍有问题，请联系管理员

### Q2：考试中途可以暂停吗？

**A**：系统支持中途休息，但建议一次性完成考试。刷新页面会丢失当前答题进度。

### Q3：提交后可以修改答案吗？

**A**：提交后无法修改答案，请确认后再提交。

### Q4：如何导出学习报告？

**A**：在结果页面点击「导出学习报告」按钮，系统会自动生成PDF文件供下载。

### Q5：前沿拓展加载失败怎么办？

**A**：前沿拓展需要联网获取，请检查网络连接后重试。

### Q6：支持手机端访问吗？

**A**：支持手机端访问，页面已适配移动端显示。
