# 🔧 微信小程序开发调试指南

## 当前问题：登录失败

### 问题现象
- 模拟器提示："登录失败，请重试"
- 调试面板显示有错误和警告

---

## 💡 解决方案

### 方案一：关闭域名校验（开发阶段必须）

**在微信开发者工具中**：

1. **点击右上角"详情"按钮**

2. **选择"本地设置"标签**

3. **勾选以下选项**：
   - ✅ **不校验合法域名、web-view（业务域名）、TLS版本以及HTTPS证书**
   - ✅ **不校验安全域名、TLS版本以及HTTPS证书**

4. **重新编译**：
   - 点击顶部"编译"按钮
   - 或按快捷键：Ctrl+B（Windows）/ Command+B（Mac）

---

### 方案二：查看具体错误信息

**打开调试面板**：

1. **点击底部"Console"标签**

2. **查看红色错误信息**，可能看到：
   ```
   request:fail url not in domain list
   ```
   或
   ```
   request:fail 
   ```

3. **截图或复制错误信息给我**

---

### 方案三：检查网络请求

**打开 Network 面板**：

1. **点击底部"Network"标签**

2. **重新点击登录按钮**

3. **查看网络请求**：
   - 红色表示失败
   - 点击失败的请求查看详情

---

## ⚠️ 开发环境 vs 生产环境

### 开发环境（当前）
- 可以关闭域名校验
- 可以使用任何域名
- 仅在开发者工具中有效

### 生产环境（上线后）
- 必须配置合法域名
- 域名必须：
  - HTTPS协议
  - 已备案
  - 在小程序后台配置
